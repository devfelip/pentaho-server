# Pentaho Server Docker Container

### Requirements
- Docker;
- Docker Compose (Optional).

## Run application
### Docker-Compose
- Clone repository;
- Open folder pentaho-server-docker in terminal:
```
cd pentaho-server-docker
```
- Execute application:
```
sudo docker-compose up -d
```
- Open app in the browser:
```
http://localhost:8001
```
